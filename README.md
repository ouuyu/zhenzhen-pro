# AI 聊天服务

## 模型选择功能

用户可以在查询开头指定不同的模型，系统会自动识别并使用相应的模型进行回答。

### 支持的模型简称

| 简称   | 对应模型                |
| ------ | ----------------------- |
| ds     | deepseek-ai/DeepSeek-V3 |
| gemini | gemini-2.0-flash        |
| claude | claude-3-5-sonnet       |
| gpt4   | gpt-4o                  |

### 使用方法

在查询开头添加模型简称，格式为：`模型简称: 你的问题`

如果不指定模型，系统将使用默认模型。

### 返回结果

当使用特定模型时，返回结果中会包含所使用的模型信息。
